htmldiff docker image (minidocks/htmldiff)
=========================================

Diff and markup HTML with `<ins>` and `<del>` tags.

Utilities
---------

- githtmldiff

Usage
-----

```bash
docker run --rm -it -v `pwd`:/app -w /app minidocks/htmldiff htmldiff --help
```
